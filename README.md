# vim-weather

## Usage
`:Weather [city]`

## Installation
To install using [vim-plug](https://github.com/junegunn/vim-plug):
```
Plug 'yorimoi/vim-weather'
```
`:PlugInstall`

## Requirements
Vim version 8.0 or higher.  
And you need the \`[curl](https://curl.haxx.se/)\` or \`[wget](https://www.gnu.org/software/wget/)\` command.
